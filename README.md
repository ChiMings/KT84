# KT68 动画工坊 (Animation Studio)

> 专为 Machenike KT68 智能键盘打造的 LED 像素屏动画设计工具。
> 极简设计，强大功能，轻松创造赛博朋克风格的扫描文字特效。

---

## 核心功能

*   **可视化设计**：通过优雅的网页界面即可设计动画。
*   **自由定制**：支持自定义文字内容、渐变色彩、高亮扫描线以及背景氛围特效。
*   **智能算法**：内置物理光效模拟引擎，自动处理亚像素级平滑扫描与粒子故障效果（Glitch）。
*   **自动适配**：智能帧率控制系统会自动优化动画时长，确保完美适配 KT68 硬件的 312 帧上限。
*   **极简体验**：采用 Apple 设计语言打造的 UI，所见即所得的实时预览。

---

## 使用指南

1.  找到文件夹中的 `led_studio.html`。
2.  双击使用任意现代浏览器（Chrome / Edge / Safari）打开。
3.  在控制面板输入你想显示的文字，调节你喜欢的配色。
4.  点击 "导出配置文件"，即可获得 `.json` 文件。
5.  在 KT68 官方驱动软件中导入该文件，即可点亮你的键盘。
